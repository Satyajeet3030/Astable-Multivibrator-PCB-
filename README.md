
# Astable Multivibrator PCB Design

Overview
This project is a PCB design of an Astable Multivibrator created using KiCad.

 Features
- Two-transistor astable multivibrator
- Through-hole components
- Single-layer PCB
- Compact board layout

 Software
- KiCad 9

 Files Included
- Schematic
- PCB Layout
- Gerber Files
- Drill Files
